Zuul is a JVM-based router and server-side load balancer from Netflix.

Netflix uses Zuul for the following:

    * Authentication
    * Insights
    * Stress Testing
    * Canary Testing
    * Dynamic Routing
    * Service Migration
    * Load Shedding
    * Security
    * Static Response handling
    * Active/Active traffic management
